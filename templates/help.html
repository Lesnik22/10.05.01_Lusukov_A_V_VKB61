Помощь и FAQ

Ответы на часто задаваемые вопросы

Быстрый старт

  1. Зарегистрируйтесь и выберите свой отдел
  2. Войдите в систему
  3. Настройте MFA для дополнительной безопасности
  4. Загрузите файлы и выберите уровень доступа
  5. Делитесь файлами с коллегами

  1. Перейдите на главную панель (Dashboard)
  2. В секции "Загрузить файл" нажмите "Выберите файл"
  3. Выберите уровень доступа:
    • Приватный - только вы
    • Отдел - коллеги из вашего отдела
    • Публичный - все пользователи
  4. При выборе "Отдел" укажите название отдела
  5. Нажмите "Загрузить"
Максимальный размер файла: 100MB

  1. Войдите в систему
  2. Нажмите на свое имя пользователя в правом верхнем углу
  3. Выберите "Настройка MFA"
  4. Отсканируйте QR-код в приложении аутентификатора
  5. Введите 6-значный код для подтверждения

Рекомендуемые приложения:

  • Google Authenticator (Android/iOS)
  • Microsoft Authenticator (Android/iOS)
  • Authy (Android/iOS/Desktop)

Для приватных файлов:

  1. Найдите файл в таблице "Мои файлы"
  2. Нажмите кнопку "Поделиться" (зеленая иконка)
  3. Введите имя пользователя
  4. Выберите уровень разрешения:
    • Только чтение - может скачать
    • Чтение и запись - может изменять
    • Полный доступ - может удалять
  5. Нажмите "Предоставить доступ"
Примечание: Для публичных файлов предоставление доступа не требуется - они автоматически доступны всем пользователям.

Система отделов позволяет автоматически предоставлять доступ к файлам коллегам из вашего отдела.

Как это работает:

  • При регистрации выберите свой отдел (IT, HR, Finance и т.д.)
  • При загрузке файла выберите уровень доступа "Отдел"
  • Укажите отдел для файла
  • Все пользователи этого отдела автоматически увидят файл
Совет: Используйте уровень "Отдел" для документов, которые должны быть доступны всей команде без необходимости предоставлять доступ каждому пользователю отдельно.

Для владельца файла:

  1. Найдите файл в таблице "Мои файлы"
  2. Нажмите красную кнопку "Удалить"
  3. Подтвердите действие

Для администратора:

  1. Перейдите в "Админ панель"
  2. В разделе "Файлы" найдите нужный файл
  3. Нажмите красную кнопку "Удалить"
  4. Подтвердите действие
Внимание: Удаление файла необратимо! Файл будет полностью удален из системы и с диска.

Обратитесь к администратору системы для сброса пароля.

Администратор может:

  • Перейти в "Админ панель"
  • Найти вашего пользователя
  • Нажать "Редактировать"
  • Установить новый пароль

Да, ваши файлы защищены несколькими уровнями безопасности:

Шифрование:
  • Все файлы шифруются при загрузке
  • Используется AES-128
  • Расшифровка только при скачивании
Контроль доступа:
  • Ролевая модель доступа
  • Явное предоставление прав
  • Полный аудит действий
Гарантия: Даже при физическом доступе к серверу файлы остаются зашифрованными и недоступными для чтения.

Полезные советы

  • Игнорировать настройку MFA
