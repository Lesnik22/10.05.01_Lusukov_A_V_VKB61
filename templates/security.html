Многофакторная аутентификация (MFA)

Двойная защита вашего аккаунта

MFA требует не только пароль, но и временный код из приложения аутентификатора на вашем смартфоне. Даже если кто-то узнает ваш пароль, без доступа к вашему телефону войти не получится.

  • TOTP: Одноразовые коды на основе времени
  • Обновление каждые 30 секунд
  • Поддержка Google Authenticator, Authy, Microsoft Authenticator

Шифрование файлов

Защита данных на уровне файлов

Каждый загруженный файл автоматически шифруется с использованием криптографической библиотеки Fernet.

  • Алгоритм: AES-128 в режиме CBC
  • Аутентификация: HMAC для проверки целостности
  • Автоматическая генерация IV
  • Расшифровка только при скачивании
Важно: Файлы хранятся на диске в зашифрованном виде. Даже при физическом доступе к серверу данные остаются защищенными.

Ролевая модель доступа (RBAC)

Три уровня ролей:
Роль Разрешения Описание
Admin Все права Полный доступ к системе, управление пользователями, удаление любых файлов
Manager Загрузка Скачивание Предоставление доступа Управление файлами и предоставление доступа другим пользователям
User Загрузка Скачивание Базовые права: загрузка и скачивание файлов

Уровни доступа к файлам

Private

Только владелец или пользователи с явным доступом

Department

Все пользователи того же отдела

Public

Все пользователи системы
